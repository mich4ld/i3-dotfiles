### Requirements:
- You need i3-gaps

### Install Polybar from AUR:
Example with `paru`:
```
$ paru -S polybar
```

### Install with script
```bash
$ git clone https://github.com/michaldzi/dotfiles.git
$ cd dotfiles
$ chmod +x ./install.sh
$ ./install.sh
```
